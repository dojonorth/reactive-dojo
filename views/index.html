Reactive Programming Dojo

Today we're going to be doing a bit of coding in the realm of functional reactive programming. You already seem to have the node application up and running, which is encouraging.

This is a bit of a 'full-stack' dojo in that we shall be messing about on both the front end and the back end, however, we shall try to keep everything as technology neutral as possible. I'll also try to err away from anything too esoteric and keep us focussed on principles rather than specific implementations.

If you are curious, the library we're using to create and manipulate event streams is batman.js. Feel free to check out the project page if you want to see how and why everything plugs together.
